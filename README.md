# JPEG Viewer with Progressive Loading

Qt-приложение для загрузки и сохранения JPEG изображений с поддержкой прогрессивной записи.

## Возможности

- Загрузка JPEG изображений
- Поддержка прогрессивной записи:
  - Автоматическое определение прогрессивных JPEG
  - Показ первого приближения при загрузке
  - Кнопка ">" для загрузки следующих приближений
  - Кнопка становится неактивной после последнего приближения
- Сохранение JPEG с настройками:
  - Флажок для режима прогрессивной записи
  - Выпадающий список для выбора режима DCT (Integer/Fast/Float)
  - Ползунок качества от 0 до 100

## Паттерны проектирования

1. **Strategy Pattern** (`jpegstrategy.h/cpp`) - различные стратегии загрузки и сохранения JPEG
2. **Factory Pattern** (`imagehandler.h/cpp`) - создание обработчиков изображений
3. **Observer Pattern** (`jpegloader.h`) - уведомления о загрузке изображений
4. **Command Pattern** (`jpegloader.h`, `jpegsaver.h`) - инкапсуляция операций загрузки и сохранения

```bash
./jpeg_viewer
```

## Требования

- Qt 6.x
- C++17
- qmake

## Примечания

В текущей реализации используется `QImage::save()` для сохранения JPEG с настройкой качества. Полная поддержка прогрессивной записи и DCT методов в Qt6 может потребовать использования дополнительных библиотек (libjpeg/libjpeg-turbo) или настройки плагинов Qt. Интерфейс приложения поддерживает все настройки, но некоторые опции могут быть ограничены возможностями Qt6.

## Использование

1. Нажмите "Load JPEG" для загрузки изображения
2. Если изображение прогрессивное, используйте кнопку ">" для загрузки следующих приближений
3. Настройте параметры сохранения (качество, DCT метод, прогрессивная запись)
4. Нажмите "Save JPEG" для сохранения изображения
